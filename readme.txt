== Description ==

WP Affiliate Link Manager add the links to your keywords based on the selection being made , making it easier to add the link to the keywords and generate the revenue via the affiliate marketing.

Major features in WP Affiliate Link Manager  include:

* can add custom link to any keyword in the website
* seperate space for linking to posts / pages content
* Easy to use UI in the backend
* No Limit for the keywords and the links to be added
